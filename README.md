# 🎙️ Voice AI: Premium Interview Prep

Voice AI is a state-of-the-art, AI-powered platform designed to help candidates master their job interviews. Built with a focus on real-time interaction and deep performance insights, it provides a seamless and "premium" experience for practicing interviews.

![Voice AI Hero](/public/hero-preview.png) *(Note: Add a real screenshot here)*

## ✨ Features

- **Real-time AI Interviewer**: Powered by **Vapi.ai**, experience natural, voice-based conversations with an AI that mimics a real interviewer.
- **Instant AI Feedback**: After every interview, receive a detailed performance analysis generated by **Google Gemini (AI SDK)**.
- **Categorized Scoring**: Get scores on Communication, Technical Knowledge, Problem-Solving, Cultural Fit, and Confidence.
- **Interview History**: Keep track of all your past sessions and monitor your progress over time.
- **Premium UI/UX**: A stunning "Midnight Violet" theme with modern typography (**Outfit**) and smooth glassmorphism effects.
- **Secure Authentication**: Robust sign-in and sign-up flows powered by **Firebase Auth**.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **AI/Voice**: [Vapi.ai](https://vapi.ai/) & [Google Gemini](https://ai.google.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Backend/Database**: [Firebase](https://firebase.google.com/) (Auth, Firestore, Admin SDK)
- **Forms**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
- **Typography**: [Outfit](https://fonts.google.com/specimen/Outfit)

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- A Firebase Project
- A Vapi.ai Account
- A Google AI (Gemini) API Key

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/chandani7021/AI-Prep.git
   cd voice-ai
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env.local` file in the root directory and add the following:
   ```env
   # Firebase (Client)
   NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
   NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_auth_domain
   NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
   
   # Firebase (Admin)
   FIREBASE_PROJECT_ID=your_project_id
   FIREBASE_CLIENT_EMAIL=your_client_email
   FIREBASE_PRIVATE_KEY="your_private_key"

   # Vapi
   NEXT_PUBLIC_VAPI_PUBLIC_KEY=your_vapi_key
   NEXT_PUBLIC_VAPI_WORKFLOW_ID=your_workflow_id

   # AI SDK
   GOOGLE_GENERATIVE_AI_API_KEY=your_gemini_key
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **Open the app**:
   Navigate to [http://localhost:3000](http://localhost:3000) in your browser.
